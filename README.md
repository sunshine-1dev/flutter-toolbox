# 🧰 Flutter Toolbox

一个实用的 Flutter 生活工具箱 App，包含 16+ 常用工具。

## ✨ 功能列表

### 🔢 计算工具
- **计算器** - 标准科学计算器
- **单位换算** - 长度、重量、温度、面积、体积
- **汇率换算** - 主要货币实时换算
- **BMI 计算器** - 身体质量指数计算
- **房贷计算器** - 等额本息/等额本金

### 📝 文本工具
- **二维码工具** - 生成和扫描二维码
- **字数统计** - 中英文字数、字符数、行数统计
- **JSON 格式化** - JSON 美化和压缩
- **文本加解密** - Base64、MD5、URL 编码解码

### 🔧 生活工具
- **手电筒** - 调用闪光灯
- **指南针** - 方向指示
- **噪音检测** - 环境分贝测量
- **颜色提取器** - 调色板 + HEX/RGB
- **倒计时/秒表** - 计时工具
- **随机数生成器** - 自定义范围
- **密码生成器** - 可定制长度和复杂度

## 🚀 开始使用

```bash
# 克隆项目
git clone https://github.com/sunshine-1dev/flutter-toolbox.git
cd flutter-toolbox

# 安装依赖
flutter pub get

# 运行
flutter run
```

## 📦 构建 APK

```bash
flutter build apk --release
```
